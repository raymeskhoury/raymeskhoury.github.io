Try out the functions using the following syntax:

=BINANCE.CONVERT("btc", NOW(), 10)

Where:

  • The currency code is the currency to convert from into AUD.
  • The date/time is the date/time of the transaction in UTC.
  • The last parameter is the amount of the currency to convert.

USDT is always used as an intermediate currency and assumed to be equal to 1USD. USD to AUD data is pulled on a per-day average basis from ozforex.com. Crypto data is pulled from binance klines on a per-hour basis. If the conversion time falls exactly on the hour, the kline open price is used. If it falls mid-hour, the average of the high and low for the hour is taken.

The following functions are also provided:

=BINANCE.EXTRACT_AMOUNT("-4000.35 BTC") =BINANCE_EXTRACT_CURRENCY("-4000.35 BTC")

EXTRACT_AMOUNT will return -4000.35 in this case, i.e. the amount of the currency. EXTRACT_CURRENCY will return BTC in this case, i.e. the currency type.
